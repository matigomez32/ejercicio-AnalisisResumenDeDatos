Análisis de Ventas 2026
Proyecto desarrollado en Python que realiza un análisis completo de datos de ventas utilizando estructuras de datos como listas y diccionarios.
¿Qué hace el programa?

Carga una lista de ventas con fecha, producto, cantidad y precio
Calcula los ingresos totales generados
Identifica el producto más vendido
Calcula el precio promedio de venta por producto
Agrupa los ingresos por día
Genera un resumen completo por producto

Tecnologías usadas

Python 3

Cómo ejecutarlo
python ventas_2026.py
